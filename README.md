# Личный кабинет ученика (Streamlit)

Веб-приложение показывает оценки конкретного ученика по вводу фамилии и пароля.

## Новое
- Крупные поля ввода и крупный шрифт в таблице.
- Поддержка загрузки файла `Students.xlsx` по публичной ссылке Яндекс.Диска (переменная `STUDENTS_PUBLIC_URL` или поле в интерфейсе).

## Структура
```
app/
  app.py
  data/
    auth.xlsx        # таблица авторизации
    Students.xlsx    # таблица с оценками (локально, если не используете Я.Диск)
  requirements.txt
  README.md
```

## Формат
- `auth.xlsx`: колонки `Фамилия`, `Пароль`.
- `Students.xlsx`: колонки `Фамилия`, `Имя` + любые даты/оценки.
- Названия колонок нормализуются (регистр/пробелы не критичны).

## Запуск локально
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Яндекс.Диск
1) В `app.py` укажите `STUDENTS_PUBLIC_URL = "https://disk.yandex.ru/d/XXXXXXXX"` **или** вставьте ссылку в поле внутри приложения (экспандер «Источник данных...»).  
2) Приложение само запросит API Яндекс.Диска и скачает файл при каждом запуске/обновлении кэша.
